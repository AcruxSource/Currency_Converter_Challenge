## Descriptcion:

**Challenge: Conversor de monedas** empleando **JAVA**

El proyecto consiste en un conversor de divisas desarrollado en Java, diseñado para facilitar a los usuarios la conversión entre distintas monedas de manera eficiente y sencilla. Este conversor soporta múltiples monedas de uso común, suministrada por https://www.exchangerate-api.com/.

Su interfaz es intuitiva y amigable, permitiendo a los usuarios efectuar conversiones rápidamente. Además, integra una base de datos que se actualiza constantemente con las tasas de cambio más recientes, asegurando así la precisión y actualidad de las conversiones.

Una característica destacada del proyecto es la funcionalidad de conversiones históricas, que posibilita a los usuarios acceder a las tasas de cambio de fechas pasadas. Esto resulta especialmente útil para quienes requieren elaborar informes financieros o desean monitorear la evolución del valor de las monedas a través del tiempo.

En resumen, este conversor de divisas es una herramienta valiosa y funcional para quienes necesitan convertir monedas frecuentemente. Gracias a su interfaz intuitiva y su base de datos actualizada, se presenta como una solución fiable y precisa para usuarios de cualquier nivel de experiencia.
